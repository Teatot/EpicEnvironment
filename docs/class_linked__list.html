Linked_list::Linked_list ( std::vector< int > lst_values)

Constructor Creates a linked list object from the values stored in the vector object.

Each value will be assigned a node, and all the nodes will form the linked list. The formation of the linked list object will maintain the same order as seen in the vector object.

Parameters
lst_valuesThe values to assign the nodes.

◆ pop_front()

void Linked_list::pop_front ( )

Method will remove the current head node and replace it with the next node in the list, if there is any.

If the linked list is empty, nothing will happen

Example

Linked_list lst = Linked_list();
lst.push_front( 3 );
lst.push_front( 4 );
std::cout << lst.front() << std::endl;
lst.pop_front();
std::cout << lst.front() << std::endl;
